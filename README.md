# Customer Churn Prediction using Machine Learning

## 📌 Overview
This project aims to predict whether a customer will leave a service (churn) using machine learning techniques. It uses a real-world telecom dataset and applies data preprocessing, model training, and evaluation.

## 🚀 Features
- Data preprocessing and cleaning
- Feature selection and analysis
- Machine Learning model using Random Forest
- Model evaluation using accuracy and confusion matrix
- Data visualization using graphs
- Real-time prediction using user input

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Visual Studio Code

## 📊 Dataset
- Telco Customer Churn Dataset
- Contains customer details like tenure, monthly charges, total charges, etc.

## ⚙️ How to Run
1. Install required libraries:
   pip install pandas scikit-learn matplotlib

2. Run the project:
   python main.py

## 🎯 Output
- Predicts whether a customer will churn or not
- Displays accuracy and confusion matrix
- Shows data visualization graph
